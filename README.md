# CineMatch

CineMatch is a fullstack movie application where users can browse movies, view details, and manage a personal watchlist.

---

## Features

- Browse movies
- Search, filter, and sort
- View detailed movie information
- Add and remove movies from a watchlist
- Data persisted via backend API

---

## Tech Stack

Frontend:

- React
- TypeScript
- Vite

Backend:

- Spring Boot
- PostgreSQL

---

## Project Structure

src/
├── api/ API calls (movies, auth, watchlist)
├── components/ Reusable UI components
├── pages/ Application pages
├── types/ Shared TypeScript types
├── utils/ Helper functions

The project uses shared types and utilities to avoid duplication and keep logic consistent.
